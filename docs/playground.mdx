# 플레이그라운드

플레이그라운드를 사용하면 **아무것도 설치하지 않고 브라우저에서 Docusaurus를 실행**할 수 있습니다!

주로 다음과 같은 용도로 유용합니다:

- Docusaurus 테스트
- 버그 보고

쉽게 기억할 수 있는 링크인 [docusaurus.new](https://docusaurus.new)를 사용하세요.

아래에서 사용 가능한 옵션 중 하나를 선택하세요.

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

편의를 위해, 다음에 [docusaurus.new](https://docusaurus.new)를 방문할 때 여러분의 선택을 기억할 것입니다.

:::
